# 个人文件仓库管理系统 — 需求文档

## 核心理念

**不碰源文件、手工入库、集中管理** 的本地文件管理工具。

- 不移动或复制源文件，仅记录文件在硬盘上的原始路径
- 手动选择需要管理的文件，而非全盘扫描
- 所有记录集中存储在单个 SQLite 数据库中

---

## 功能清单

### 1. 文件管理

| 功能 | 说明 |
|------|------|
| 添加文件 | 通过原生文件选择对话框（多选）手动添加文件到仓库，自动获取文件名、大小、扩展名等信息 |
| 批量添加 | 一次选择多个文件入库，支持统一指定分类和备注 |
| 编辑文件 | 修改文件标题、分类、备注、评分 |
| 删除文件 | 从仓库中移除记录（不删除源文件），支持单个/批量删除 |
| 打开文件 | 使用系统默认程序打开源文件 |
| 定位文件 | 在资源管理器中定位到文件所在位置 |
| 文件校验 | 校验文件是否仍然存在于原始路径，标记丢失文件 |

### 2. 文件夹（分类）管理

| 功能 | 说明 |
|------|------|
| 新建文件夹 | 自定义名称和颜色标识 |
| 编辑文件夹 | 修改名称和颜色 |
| 删除文件夹 | 删除后，其中文件自动变为未分类状态 |
| 侧边栏展示 | 显示文件夹列表及各文件夹文件数量 |

### 3. 浏览与筛选

| 功能 | 说明 |
|------|------|
| 全部文件 | 展示仓库中所有文件 |
| 现存文件 | 仅显示文件路径仍然有效的记录 |
| 已丢失 | 仅显示文件已被移动或删除的记录 |
| 分类筛选 | 按文件夹分类过滤 |
| 关键词搜索 | 搜索文件名、路径、备注等字段 |
| 排序 | 按添加时间、更新时间、名称、大小、评分排序 |

### 4. 数据管理

| 功能 | 说明 |
|------|------|
| 导出 JSON | 将全部数据导出为 JSON 文件，方便备份 |
| 导入 JSON | 从 JSON 文件恢复数据 |

### 5. 统计面板

- 总文件数、总大小
- 现存/丢失文件数量
- 近 7 天新增文件数

## 技术架构

| 层级 | 技术选型 |
|------|---------|
| 后端 | Python Flask |
| 数据库 | SQLite（单文件，零配置） |
| 前端 | 原生 HTML/CSS/JS（单页面，无框架依赖） |
| 桌面窗口 | pywebview（原生桌面窗口，不依赖浏览器） |
| 文件选择 | tkinter filedialog（获取真实绝对路径） |
| 打包分发 | PyInstaller（打包为单个 exe） |

## 数据模型

### categories 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| name | TEXT UNIQUE | 分类名称 |
| color | TEXT | 颜色标识 |
| created_at | TEXT | 创建时间 |

### files 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 主键 |
| title | TEXT | 显示标题 |
| file_path | TEXT | 文件原始路径 |
| file_name | TEXT | 文件名 |
| file_ext | TEXT | 扩展名 |
| file_size | INTEGER | 文件大小（字节） |
| category_id | INTEGER FK | 所属分类 |
| tags | TEXT | 标签 |
| notes | TEXT | 备注 |
| rating | INTEGER | 评分（0-5） |
| created_at | TEXT | 入库时间 |
| updated_at | TEXT | 最后更新时间 |
| last_checked | TEXT | 最后校验时间 |
| file_exists | INTEGER | 文件是否存在（1/0） |

## 设计原则

- **零侵入**：不修改、不移动用户的源文件
- **单文件数据库**：所有数据存在一个 `.db` 文件中，备份/迁移只需复制一个文件
- **无框架前端**：纯原生 JS，零依赖，加载快
- **本地优先**：无需网络，数据完全掌控在用户手中
- **桌面原生**：使用 pywebview 在独立窗口中运行，无需打开浏览器
- **可分发**：支持通过 PyInstaller 打包为单个 exe 文件，分发即用
